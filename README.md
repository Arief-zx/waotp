# waotp

Bot Telegram **OTP WhatsApp** gratis, real-time, dan multi-negara.  
Dapatkan kode OTP WhatsApp dari nomor virtual smsonline.cloud yang selalu update, serta fitur pencarian nomor mudah & tombol interaktif!

---

## ✨ Fitur Utama

- **Realtime OTP WhatsApp**  
  Kirim OTP WhatsApp terbaru langsung ke channel Telegram, update otomatis setiap detik!

- **Summary Nomor Virtual**  
  Daftar lengkap nomor virtual dari berbagai negara, dikirim ke grup & channel tiap 2 jam.

- **Tombol Interaktif**  
  Pilih negara, ganti nomor acak, dan dapatkan link OTP dengan sekali klik di chat bot.

- **Anti Delay**  
  Begitu OTP WhatsApp muncul di web, langsung dikirim ke channel Telegram.

- **Gratis & Opensource**  
  Script Python open source, mudah deploy di Railway dan VPS.

---

## 🚀 Deploy ke Railway

1. **Fork** repo ini & clone ke PC kamu.
2. Edit/isi file `.env` atau atur environment variable di Railway:  
   - `TOKEN` = Token Bot Telegram
   - `CHANNEL_CHAT_ID` = ID channel (cth: `-1002964625223`)
   - `GROUP_CHAT_ID` = ID grup (cth: `-1002825740190`)
3. Deploy melalui Railway (deploy from GitHub repo).
4. Done! Bot siap pakai.

---

## 🤖 Cara Pakai

- Ketik `/start` di chat bot (bukan di grup/channel)
- Gunakan tombol **Country** untuk pilih negara nomor
- Klik **Change** untuk mendapatkan nomor random
- Klik **GetOTP** untuk cek kode OTP terbaru di channel

---

## 🛡️ Disclaimer

- Bot ini hanya menampilkan OTP dari layanan publik smsonline.cloud.
- Jangan gunakan untuk aktivitas ilegal!
- Kode OTP bisa diakses siapa saja (open number), **tidak privat**.
